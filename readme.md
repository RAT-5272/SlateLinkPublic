## What is Slatelink
Slatelink is a messaging client I created to interface with the SlateOS messaging servers, basically my own version of messaging software allowing you to communicate with other users and users of SlateOS. It was also created to have good* privacy; it forces HTTPS for encrypted communication, and the message contents are further encrypted before being sent and after being received, meaining the server only stores encrypted messages.
